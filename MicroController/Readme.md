# CS7.2 Firmware

## Installing the firmware
1. Install Arduino Studio
2. Download the latest firmware zip file from this folder
3. Extract the zipfile and navigate into the folder and double-click on the CS72_Firmware.ino file to launch the file in the studio
4. In Arduino Studio, open the library panel (the icon on the left side which looks like books) and search for TMCStepper
5. Install the TMCStepper Library
6. Identify which COM port your board is plugged into. YOu can turn off the board and check the list of ports, then turn it back on. The new one that appears will be the com port for your board
7. Select the board/com port and set the board type as Arduino Uno
8. Push/Upload the firmware to the boards.  This is the button in the top left of the studio which looks like an arrow.
